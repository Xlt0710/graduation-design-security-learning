# AI辅助的Web安全学习与漏洞靶场训练系统

## 1. 项目定位

本系统面向网络安全初学者，提供 Web 安全知识学习、漏洞靶场练习、在线提交答案、学习进度记录、AI 个性化推荐、AI 漏洞报告辅助生成等功能。系统采用前后端分离架构，前端使用 Vue，后端使用 Spring Boot，数据库使用 MySQL。

项目重点不是做真实攻击平台，而是做一个安全、可控、面向教学和毕业设计展示的 Web 安全学习系统。靶场内容以内置题目和受控环境为主，例如 SQL 注入、XSS、弱口令、文件上传漏洞等。

## 2. 主要用户角色

| 角色 | 说明 |
| --- | --- |
| 普通用户 / 学生 | 注册登录后学习课程、完成靶场练习、查看推荐内容和学习报告 |
| 管理员 / 教师 | 管理用户、课程、漏洞题目、靶场环境、公告、系统数据 |
| AI 服务 | 根据用户学习行为生成推荐、学习建议和漏洞报告草稿 |

## 3. 功能模块

### 3.1 用户与权限模块

用户可以注册、登录、退出登录、修改个人资料、修改密码。系统根据用户角色控制访问权限，普通用户只能访问学习和练习功能，管理员可以进入后台管理页面。

核心功能：

- 用户注册与登录
- JWT 登录认证
- 用户角色权限控制
- 个人信息维护
- 学习数据绑定到用户账号

### 3.2 首页与仪表盘模块

用户登录后进入个人仪表盘，查看学习进度、最近练习记录、AI 推荐课程、AI 推荐靶场题目、错题和薄弱知识点。

核心功能：

- 学习进度统计
- 最近练习记录
- 推荐学习内容
- 推荐靶场题目
- 漏洞类型掌握情况统计

### 3.3 课程学习模块

系统提供 Web 安全基础课程，包括 SQL 注入、XSS、CSRF、文件上传、认证与会话安全、弱口令、越权漏洞等内容。每个课程可以包含多个章节，每个章节包含文字、图片、代码示例、测试题。

核心功能：

- 课程列表
- 课程详情
- 章节学习
- 学习进度记录
- 章节测验
- 收藏课程

### 3.4 靶场练习模块

系统提供受控的 Web 漏洞练习题。用户进入题目后查看题目描述、漏洞类型、难度、提示信息，在本地或系统内置靶场页面中完成练习，最后提交 flag 或答案。

核心功能：

- 靶场题目列表
- 按漏洞类型、难度筛选
- 题目详情
- 查看提示
- 提交 flag / 答案
- 自动判题
- 练习记录保存
- 错题回顾

### 3.5 AI 推荐模块

AI 推荐模块根据用户的学习进度、练习正确率、漏洞类型掌握情况、题目难度完成情况，推荐下一步学习内容和靶场题目。

推荐逻辑可以先采用规则算法，后续再接入大模型生成自然语言学习建议。

推荐依据：

- 用户未学习课程
- 用户错题较多的漏洞类型
- 用户已掌握的难度等级
- 最近练习频率
- 课程和题目的标签匹配

核心功能：

- 推荐课程
- 推荐靶场题目
- 生成学习建议
- 生成阶段性学习计划
- 记录推荐结果

### 3.6 AI 漏洞报告辅助模块

用户完成靶场练习后，可以填写漏洞复现步骤、影响分析、修复建议，系统调用 AI 辅助生成规范化漏洞报告草稿。该功能适合毕业设计展示“AI + 网络安全”的结合。

核心功能：

- 用户填写漏洞信息
- AI 生成报告草稿
- 用户编辑报告
- 保存报告
- 导出报告

报告结构：

- 漏洞名称
- 漏洞类型
- 风险等级
- 漏洞描述
- 复现步骤
- 影响范围
- 修复建议
- 参考资料

### 3.7 学习记录与统计模块

系统记录用户课程学习、题目练习、测验成绩、报告生成等行为，并生成可视化统计结果。

核心功能：

- 学习时长统计
- 课程完成率
- 题目完成率
- 漏洞类型正确率
- 每周学习趋势
- 个人学习报告

### 3.8 后台管理模块

管理员可以维护系统基础数据，包括用户、课程、章节、题目、标签、公告和系统配置。

核心功能：

- 用户管理
- 角色管理
- 课程管理
- 章节管理
- 靶场题目管理
- 标签管理
- 公告管理
- 数据统计

## 7. 推荐算法初版设计

为了降低实现难度，第一版推荐算法可以使用规则评分，不一定一开始就训练机器学习模型。

推荐课程评分示例：

| 条件 | 加分 |
| --- | --- |
| 用户未学习该课程 | +20 |
| 课程标签匹配用户错题漏洞类型 | +30 |
| 课程难度等于用户当前水平 | +20 |
| 课程被较多用户学习 | +10 |

推荐靶场评分示例：

| 条件 | 加分 |
| --- | --- |
| 用户未完成该题 | +30 |
| 题目漏洞类型属于用户薄弱项 | +30 |
| 题目难度适合用户当前水平 | +20 |
| 用户最近学习过相关课程 | +20 |

AI 可以负责把推荐结果转换成更自然的解释，例如：“你最近 SQL 注入题目的正确率较低，建议先学习 SQL 注入基础课程，再完成 2 道简单难度题目。”

## 9. 非功能需求

| 类型 | 要求 |
| --- | --- |
| 安全性 | 密码加密存储，接口鉴权，管理员接口权限控制 |
| 可用性 | 页面操作简单，适合初学者使用 |
| 可维护性 | 前后端分离，接口结构清晰 |
| 性能 | 常用列表接口支持分页查询 |
